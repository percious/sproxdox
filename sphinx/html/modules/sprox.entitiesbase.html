class sprox.entitiesbase.EntitiesBase(provider_hint=None, **provider_hints)¶

Bases: sprox.viewbase.ViewBase

This view can display all of the entities for a given provider.

Modifiers :see sprox.viewbase.ViewBase

>>> class MyEntitiesBase(EntitiesBase):
...     __entity__ = User
